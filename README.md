# CareerGO Health API

Simple Go API built with Gin as the first step in my backend and AI infrastructure learning path.

## Stack
- Go
- Gin Gonic

## Run locally
```bash
go mod tidy
go run main.go
```

## Endpoint
- `GET /health`

## Response
```json
{"message":"AI Infra Go CareerGO starts!","status":"ready"}
```
